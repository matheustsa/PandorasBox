### Import pg_dump from Heroku to localhost ###
```
pg_restore --verbose --clean --no-acl --no-owner -h localhost -U <pg_user> -d <database> <heroku-dump>
```

### Change password ###
Login into the psql:

```sudo -u postgres psql```

Then in the psql console change the password and quit:

```
\password postgres
\q
```
